Web App Orçamento Pessoal

Esse web aplicativo é um pequeno projeto para testar as funcionalidade do uso do ESCM6. Foi desenvolvido utilizando as tecnologias Javascript, Html 5 e CSS 3 em conjunto com os frameworks Jquery e Bootstrap 4.

O desenvolvimento do projeto tem como base o uso do ES6 e por tanto nao possui ligaçao direta com o banco de dados, porem, usa do localStorage do browser para armazenar e recuperar os dados Poderia ser usado a sessao ou cookies, porem, como o projeto nao possui aplicaçao comercial ou pratica, optei por utilizar dessa forma.
